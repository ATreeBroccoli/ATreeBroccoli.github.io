JavaScript 预热

JavaScript简介

大概在1992年,一家称作 Nombas的公司开发了一种叫做C减减(C-minus-minus,简称Cmm)的嵌入式脚本语言.Cmm背后的理念很简单:一个足够强大可以替代宏操作(macro)的脚本语言,同时保持与C(和C++)足够的相似性,以便开发人员能很快学会.Nombas最终把Cmm名字改为ScriptEase.

在Netscape Navigator2.0即将正式发布前,Netscape 将其更名为 JavaScript,目的是为了利用Java这个因特网时髦词汇.Netscape的赌注最终得到回报,JavaScript从此变成了因特网的必备组件.

JS中的基本数据类型

JS中的变量是弱类型的,这与Java和C不同,变量的类型无特定的类型,定义变量时只用var运算符,就可以将变量初始化任意类型值.

例子:

  • Undefined 类型

Undefined类型只有一个值,即undefined.当声明的变量未初始化时,该变量的默认值是undefined.

  • Null 类型

Null类型只有一个专属值null.值undefined实际上是从值null派生来的,因此它们被定义为相等的.

  • Boolean 类型

Boolean有两个值true和false.

  • Number 类型

这种类型是比较特殊的数据类型.这种类型既可以表示32位的整数,还可以表示64位的浮点数.直接输入的任何数字都被看做Number类型.

另外整数也可以被表示为八进制或者十六进制的数.八进制数的首字母必须为0,其后的数字可以是任何八进制数字(0-7).

对于十六进制,必须以0x开头,然后是任意的十六进制数字(0-9和A-F).字母可以是大写也可以是小写.

  • String 类型

String类型一直在编程语言中都是比较有意思的类型,在JS中也是,它是唯一没有固定大小的原始类型.可以用字符串存储0或者更多的Unicode字符.
另外字符串中每一个字符都有特定的位置.首字符从位置0开始,往后依次类推.

本文只是JS的预热,之后会持续更新更加详细的JS知识.

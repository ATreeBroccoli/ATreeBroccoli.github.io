DOM

#DOM
对于HTML文档,浏览器会将其解析成为一个DOM树,要动态更改HTML的结构,就需要通过JavaScript来操作DOM.其实操作DOM节点就是就是这么几个操作:增、删、改、查.

获取节点

在操作DOM节点之前我们需要获取对应的DOM节点.

最常用的是:

document.getElementById() //通过节点的id进行获取 (唯一节点)

document.getElementsByTagName() //通过节点的标签名进行获取 (多个节点)

document.getElementsByClassName() //通过节点的class进行获取 (多个节点)

例子:

//获取id为div1的标签
var div1 = document.getElementById(“div1”);

//获取div1下的所有div标签
var divs = div1.getElementsByTagName(“div”);

divs[0]; //第一个div标签

//获取class为btn的所有标签
var btns = document.getElementsByClassName(“btn”);

btns[1]; //第二个标签

也就是说,我们可以通过的id获取对应唯一的DOM标签.通过getElementsByTagName()或者getElementsByClassName()获取一组DOM节点. 要精确地选择DOM节点,我们可以先找到父节点,然后从父节点开始选择缩小范围直到找到确定DOM节点.

另外一种做法是使用querySelector()和querySelectorAll()来进行节点的获取:

//通过querySelector() 获取选择器为head的标签

var h = document.querySelector(“#head”);

//通过querySelectorAll() 获取选择器为div .bg的所有标签

var divs = document.querySelectorAll(“div .bg”);

注意: querySelector()和querySelectorAll()在版本IE<8中不兼容,IE8仅仅有限支持

另外,严格地讲，我们这里的DOM节点是指Element，但是DOM节点实际上是Node，在HTML中，Node包括Element、Comment、CDATA_SECTION等很多种，以及根节点Document类型，但是，绝大多数时候我们只关心Element，也就是实际控制页面结构的Node，其他类型的Node忽略即可。根节点Document已经自动绑定为全局变量document。

节点关系

DOM是一个树形结构这一点大家一定要熟记,我们个个节点直接有父节点<–>子节点的关系.

获取子节点:

//获取id为div1的节点

var div1 = document.getElementById(“div1”);

//获取div1的子节点(一组)

var children1 = div1.children;

children1[0]; //第一个字节点

获取第一个和最后一个子节点:

div1.firstElementChild; //第一个节点

div1.lastElementChild; //最后一个节点

获取父节点:

//获取父节点 (唯一)

var parent1 = div1.parentNode;

获取兄弟标签:

div1.nextElementSibling; //获取下一个兄弟标签
div1.previousElmentSibling; //获取上一个兄弟标签

节点操作

创建元素:

//创建一个li标签 liTemp
var liTemp = document.createElement(“li”);

添加元素

//创建li标签
var liTemp1 = document.createElement(“li”);

var liTemp2 = document.createElement(“li”);

//在末尾添加

var ul1= document.getElementById(“ul1”);

ul1.appendChild(liTemp1);

//在目标节点前插入新的节点
ul1.insertBefore(liTemp1,liTemp2); //在liTemp1前插入liTemp2

删除元素

var ul1= document.getElementById(“ul1”);

//删除ul1中第一个子元素
ul1.removeChild(ul1.firstElementChild);

//删除ul1中最后一个子元素
ul1.removeChild(ul1.lastElementChild);

//删除对应下标的子元素
ul1.removeChild(ul1.children[2]);

替换元素

var ul1= document.getElementById(“ul1”); //获取ul1

var liTemp = document.createElement(“li”); //创建liTemp

ul1.replaceChild(liTemp,ul1.children[0]); //将liTemp替换掉ul1.children[0]
